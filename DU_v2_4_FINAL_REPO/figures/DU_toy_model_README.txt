DU Toy Model Stress Test
========================

This package computes F(C)=alpha*S_info+lambda*R_info-beta*L(M) for three coarse-grainings
on a toy binary spin-chain universe.

Files:
- DU_toy_model.py (reproducible script)
- DU_toy_model_results.csv (numeric results)
- DU_toy_model_FC.png (bar chart)

Run:
  python DU_toy_model.py
